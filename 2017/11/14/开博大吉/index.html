论第一篇博客是如何产生的

Posted on 2017-11-14

抛砖引玉

按照方方老师的课程安排，我照着Hexo教程一步步开始搭建我的第一篇博客——“开博大吉”。

“1.在 GitHub 上新建一个空 repo；
2.npm install -g hexo-cli，安装 Hexo
3.hexo init myBlog
4.cd myBlog
5.npm i
6.hexo new 开博大吉，你会看到一个 md 文件的路径
7.start xxxxxxxxxxxxxxxxxxx.md，编辑这个 md 文件，内容自己想（Ubuntu 系统用 xdg-open xxxxxxxxxxxxxxxxxxx.md 命令）
8.start _config.yml，编辑网站配置
9.把第 6 行的 title 改成你想要的名字
10.把第 9 行的 author 改成你的大名
11.把最后一行的 type 改成 type: git
12.在最后一行，与 type 平齐，加上一行 repo: 仓库地址
13.npm install hexo-deployer-git –save，安装 git 部署插件
14.hexo deploy”

步骤很清晰明了，但是我在用VSCode写md文件时总是出现“Git存储库存在大量活动更改，将仅启用部分Git功能。” 多次反复尝试还是出现同样的问题。经过方方老师指导，应该是把~纳入了git控制（其实我也不知道为什么），输入命令"rm -rf ~/.git"顺利逃脱困境！

至此完成第一篇丑丑的博客上传！
